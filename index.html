WWW.

The World Wide Web is a collection of HTML, or Hyper Text Markup Language, documents. HTML is the basis of almost all 30 billion pages on the web.

The major components of the web are Me (or you), My computer + My browser, The Internet, HTTP protocol and Servers housing the HTML documents and respond to the HTTP requests.

HTML

HTML is the heart of the web. Text content is what you see or read, but Markup is what it looks like. Plain text is plain text in HTML, but if you want to make it look different, you need to use markup/tags. HTML is also responsible for allowing references to other documents and links to other pages.

Markup Broken Down

Tags are essentially used to give direction to your content. Some tags require just an opening tag (called a void tag), while others require both an opening and closing tab. The combination of the tags and everything in between the tags is known as an element.

Tags can also have attributes. Attributes are used to insert links or images, such as one of my favorite cartoons below.
